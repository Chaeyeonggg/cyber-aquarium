## 🐟 사이버 아쿠아리움 (바이브 코딩 프로젝트)
sns의 어떤 사이버 아쿠아리움에 영감을 받고 제작하게 된 데스크탑 아쿠아리움.

🗝️ Key Features
+ Custom Pixel Art : aesprite로 직접 디자인한 픽셀 아트 에셋 적용
+ Underwater sound : 힐링이나 코딩 작업할 때 유용한 수중 소리 재생
+ Decoration Mode : 자유로운 장식 배치 및 자동 저장 기능
+ Easter Egg

🛠️ Tech Stack
+ Python
+ PyQt5
+ PyInstaller

🖼️ 에셋
+ aesprite로 직접 제작 & itch

✨ 사용한 ai
+ gemini
